# Dashboard Patterns

> Dashboard archetypes extracted from the styles catalog, focused on data presentation and admin interfaces.
> Source: UI/UX Pro Max (converted to MIKE format).

---

### 1. Data-Dense

Maximum information density with clear hierarchy. Uses compact spacing, small fonts, and subtle separators.

| Field | Value |
|-------|-------|
| Recommended Styles | Minimalism, Flat Design, Data Visualization |
| Best For | Analytics, monitoring, financial dashboards |

---

### 2. Executive

High-level KPI focus with large numbers, sparklines, and clean whitespace.

| Field | Value |
|-------|-------|
| Recommended Styles | Minimalism, Glassmorphism |
| Best For | C-suite dashboards, business intelligence |

---

### 3. Real-Time Monitoring

Live data feeds, auto-updating charts, status indicators. Optimized for dark mode.

| Field | Value |
|-------|-------|
| Recommended Styles | Dark Mode UI, Data Visualization |
| Best For | DevOps, IoT, server monitoring |

---

### 4. Admin Panel

CRUD-focused with tables, filters, and bulk actions. Sidebar navigation.

| Field | Value |
|-------|-------|
| Recommended Styles | Flat Design, Minimalism |
| Best For | CMS, user management, content moderation |

---

### 5. Analytics

Chart-heavy with interactive filtering, date range pickers, and drill-down.

| Field | Value |
|-------|-------|
| Recommended Styles | Data Visualization, Glassmorphism |
| Best For | Marketing analytics, sales reports |

---

### 6. Project Management

Kanban boards, Gantt charts, task lists. Drag-and-drop heavy.

| Field | Value |
|-------|-------|
| Recommended Styles | Flat Design, Micro-interactions |
| Best For | PM tools, issue trackers, sprint boards |

---

### 7. E-Commerce Admin

Order management, inventory, sales charts. Multi-panel layout.

| Field | Value |
|-------|-------|
| Recommended Styles | Flat Design, Minimalism |
| Best For | Store management, marketplace admin |

---

### 8. Social Media

Feed-based, engagement metrics, content calendar. Card-heavy layout.

| Field | Value |
|-------|-------|
| Recommended Styles | Soft UI Evolution, Micro-interactions |
| Best For | Social media management, community dashboards |

---

### 9. Healthcare

Patient data, vitals, appointment scheduling. HIPAA-aware design.

| Field | Value |
|-------|-------|
| Recommended Styles | Minimalism, Accessible Design |
| Best For | EHR, clinic management, telemedicine |

---

### 10. Financial

Portfolio views, transaction tables, risk indicators. Trust-focused colors.

| Field | Value |
|-------|-------|
| Recommended Styles | Minimalism, Data Visualization |
| Best For | Banking, trading, investment platforms |
