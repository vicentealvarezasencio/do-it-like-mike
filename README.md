# MIKE — Multi-profile Intelligent Kode Engine

> "Don't half ass it. Do it like Mike."

A profile-driven AI development framework for Claude Code that combines the best innovations from GSD, M2C1, PRP, BMAD, and PAUL into a unified system that adapts to your project's needs.

## Install

```bash
npx do-it-like-mike
```

## Quick Start

```
/mike:init       # Initialize your project
/mike:progress   # See what to do next
/mike:help       # Full command reference
```

## Profiles

| Profile | Best For | Process Depth |
|---------|---------|--------------|
| **BLITZ** | Quick fixes, scripts, prototypes | Minimal |
| **SPRINT** | Single features, small projects | Per-phase planning |
| **FORGE** | Multi-feature projects, team-quality code | Front-loaded planning + sharding |
| **CITADEL** | Production systems, mission-critical | Maximum rigor |
| **SCOUT** | Existing codebases, adding features | Codebase-first analysis |
| **GOLD** | Let MIKE decide | Auto-adaptive (L0-L4) |

## Pipeline

```
INITIALIZE → DISCOVER → RESEARCH → ARCHITECT → DESIGN → SHARD
→ EXECUTE → VERIFY → UNIFY → TRANSITION → COMPLETE
```

Each profile activates different steps at different depths. BLITZ uses 3 steps. CITADEL uses all 11.

## Commands

26 slash commands organized by function:

- **Pipeline:** init, discover, research, architect, design, shard, execute, verify, unify, transition, complete
- **Navigation:** progress, resume, pause
- **Planning:** discuss, plan
- **Error handling:** debug, issues
- **Codebase:** map
- **Roadmap:** add-phase, insert-phase, remove-phase
- **Utility:** quick, help, settings, update

## Key Innovations

- **Profile system** — One framework, six behaviors. From hackathon to enterprise.
- **Document sharding** — 74-90% token reduction by giving each executor only what it needs.
- **Mandatory UNIFY** — Every execution unit closes with reconciliation. No drift.
- **Dynamic rules engine** — Rules loaded just-in-time, not bloating every session.
- **Design system generation** — Industry-aware UI design from 67 styles, 96 palettes, 100 rules.
- **Context brackets** — Automatic handoff when context runs low.

## Options

```bash
npx do-it-like-mike              # Global install (default)
npx do-it-like-mike --local      # Project-local install
npx do-it-like-mike --dry-run    # Preview without installing
npx do-it-like-mike --uninstall  # Remove MIKE files
npx do-it-like-mike --force      # Skip confirmation
```
